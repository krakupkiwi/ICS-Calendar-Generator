❓ How to use — CSV format — UID explained

Standalone tool — runs entirely in your browser. No internet needed after loading. Works for any sport.

CSV format:

Date,Round,Opponent,Location,Category,Grade,Time
2026-05-02,Round 4,Wests,Home,Seniors,First Grade,15:00
2026-05-02,Round 4,Wests,Home,Womens,Womens League Tag,11:00
2026-05-09,Round 5,Norths,Away,Juniors,U/18,09:30

Category (optional) — groups rows into calendar events and forms the UID.
Examples: Seniors, Juniors, Womens, Masters, U/12s
All rows with the same Date + Category become one calendar event.
If Category is empty, the Grade field is used instead.

Stable UIDs — event IDs use your club slug + category so re-exporting updates existing subscribed calendars without creating duplicates:

fixture-2026-05-02-seniors@northbeachfccal
